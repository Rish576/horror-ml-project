# Horror Text Classification using Machine Learning

## Project Overview
This project focuses on classifying horror-related text using machine learning techniques.  
The model is trained to analyze text data and predict whether the content belongs to the horror genre.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Google Colab

## Project Workflow
- Data collection and cleaning
- Text preprocessing
- Feature extraction
- Model training and evaluation
- Prediction of horror text

## How to Run the Project
1. Open the notebook in Google Colab  
2. Run all cells step by step  
3. View the prediction results
